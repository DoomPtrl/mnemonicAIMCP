# mnemonicAIMCP
> 이 정책은 **두문자 조합 서비스(Initial-Combo API)** 및 이를 호출하는 **Custom GPT**(이하 “서비스”)에 적용됩니다.

---

## 1) 범위(Scope)
- 이 문서는 사용자가 Custom GPT에 단어 목록을 제공할 때, 해당 GPT가 Actions를 통해 호출하는 **두문자 조합 API**에서 처리되는 데이터에 관한 것입니다.  
- 이 문서는 별도의 로그인/인증 없이 사용하는 구성을 전제로 합니다.

## 2) 수집되는 정보(What We Collect)
우리는 서비스 제공을 위해 아래 정보를 처리할 수 있습니다.

- **사용자 제공 입력값**: 단어 목록(예: “결합, 근육, 상피, 신경”), 또는 초성/첫 글자 목록  
- **衍生 데이터(파생 결과)**: 조합 후보(예: ["결근", "신상"]), 점수/랭킹 및 탐색 로그(선택적)  
- **기술 메타데이터**: 요청 시각, 응답 시각, IP 주소·HTTP 헤더(서버·CDN·호스팅 로그 수준)  
- **플랫폼 처리 정보**: 사용자의 대화 내용은 OpenAI(또는 사용 플랫폼)의 인프라에서 처리·일시 보관될 수 있습니다. 해당 플랫폼의 [자체 개인정보 처리방침]을 참고하세요.

## 3) 이용 목적(How We Use Data)
- 두문자 조합 계산 및 결과 반환(서비스 제공)
- 시스템 모니터링, 오류 추적, 성능 개선(서버 로그·메트릭)
- 보안(비정상 요청 탐지·완화)

**우리는 입력 단어를 마케팅이나 광고 타겟팅에 사용하지 않습니다.**

## 4) 제3자 제공(Sharing)
- **판매하지 않음**: 개인 정보를 판매하지 않습니다.
- **처리 위탁**: 클라우드 호스팅/로그/콘텐츠 전송 네트워크(CDN)와 같은 인프라 제공업체에 처리를 위탁할 수 있습니다.
- **플랫폼 제공자**: Custom GPT 실행에는 OpenAI 등의 플랫폼이 관여합니다(대화 처리·일시 보관 등).

## 5) 보관 기간(Retention)
- 기본적으로 **최대 30일** 내에서 운영 로그 및 오류 로그를 보관할 수 있습니다(법적 의무 또는 보안 목적상 더 길어질 수 있음).  
- 조합 후보 결과는 응답 시 전달되며, 서버 측에서 장기적으로 저장하지 않도록 설계할 수 있습니다.

## 6) 쿠키 및 추적(Cookies/Tracking)
- API 자체는 쿠키를 사용하지 않습니다.
- 별도의 웹 대시보드/문서 페이지를 운영할 경우, 호스팅 제공자나 분석 도구의 쿠키/추적이 적용될 수 있습니다(선택적).

## 7) 보안(Security)
- 전송 구간 암호화(HTTPS) 등 합리적 보호조치를 적용합니다.
- 완전한 보안을 보장할 수는 없으나, 침해 최소화를 위해 모니터링과 접근 통제를 수행합니다.

## 8) 국제 이전(International Transfers)
- 서버/플랫폼 위치에 따라 데이터가 국외로 전송·저장·처리될 수 있습니다.

## 9) 이용자 권리(Your Rights)
- 접근, 정정, 삭제 등 관련 권리를 행사하려면 아래 연락처로 요청하십시오.
- 플랫폼(예: OpenAI)의 계정·대화 데이터에 관한 권리는 해당 플랫폼 정책을 따릅니다.
